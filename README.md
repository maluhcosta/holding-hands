# Holding Hands 🤝

💡 Uma aplicação para quem tem alguma ideia revolucionária e está em busca de pessoas que possam investir em sua ideia.
- Saúde;
- Educação;
- Meio ambiente;
- Igualdade de gênero;
- Causas LGBTQI+;
- Sustentabilidade;
- Outros.
